The Rough N' Tumble Dating Engine
11/09/2014

Ever wonder what is really under the hood in a dating website? Well they're not telling so I decided to build my own. Based on scientifically-proven science backed up by science, I can find your perfect match after you answer just 3 simple yes or no questions! Don't believe me. Neither do I? But here's a fun prototype anyway.

I built a class that creates user profiles and stores their information. The class has an instance method that allows each user to check their compatability based on other people's answers to the three questions. It then returns a percentage value of how compatible you are with each user!

It's still rather rough, but you can still have a pretty fun time. Feel free to test it out!
